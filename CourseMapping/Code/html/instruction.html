Instruction Page

1. Click register tab and register if you haven't registered yet!

1.a. If you have registered, please click login tab and login!

2. Now you can see the website looks like this. Please click + to create a new tab. You can create a tab up to 4, but you can always remove the tab by clicking x on the tab button.

2.a. Select a major by dropdown and press get courses button. It will give you the draggable course blocks

2.b. You can drag the courseblocks depends on your plan, and you can also change the title of the tab.

2.c. If you double-click the course block, it will show you the title and the credits of the course.

2.d. Press save if you want to save your tab. It will ask you if you want to save it or not before removing the tab.

3. If you want to see what you've saved, please click saved tab. You can navigate different saved tabs by clicking buttons on saved tab.

4. When you logout, the website will ask if you want to save your open tabs or not. If you say OK, it will save the tabs. If you say CANCEL, it won't save the tabs but still let you logout.
