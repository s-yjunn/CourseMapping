Instruction Page

1. On Courses tab, you can view, add and remove the courses from courses.json and majors.json.

2.a. You can view the course list of each department.

2.b. To select the department, click the dropdown menu and select the department you want to look at.

3.c. If you want to remove the courses, check the checkboxes and click Remove button on the top of the table.

3.d. If you remove the course from majors.json, then that course will be automatically removed from courses.json, vice versa.

4.a. Click Add Course button to add courses to courses.json and majors.json.

4.b. There are four required fields: Department, Course Number, Title, and Credit.

4.c. Example inputs: Department - ENG, Course Number - 111, Title - Introduction to English Literature, Credit - 4.

4.d. When you enter prerequisites, corequisites, required for, suggested for, and overlap, you must follow this form ENG 100, SWG 121, SDS 202S. So there should be a space between the department and the course number, a comma&a space between two courses

4.e. You can also edit information of each course using Add Course.

5. On Users tab, admin can see each user's info: username, password, server id, number of saved pathways.

6. Admin can delete all saved pathways of each user using the same method as removing the course.
